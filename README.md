# ShopIndia 🛍️ — Full Stack Ecommerce App

## Tech Stack
- **Frontend**: React 18, React Router, Axios, React Toastify
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Auth**: JWT + bcrypt
- **File Upload**: Multer

## Features
- 🛒 Buyer: Browse, search, filter by category, add to cart, buy now
- 📦 Cart animation: parcel flies into truck on "Add to Cart"
- ⭐ Product reviews & star ratings
- 🔔 Real-time notifications (offers + order alerts)
- 🏪 Seller: Add/edit/delete products with image upload
- 💰 Prices in Indian Rupees (₹)
- 📍 Full buyer address captured on order
- 📊 Seller gets full buyer details on every order

## Setup

### Prerequisites
- Node.js 18+
- MongoDB running locally (or MongoDB Atlas URI)

### Install & Run

```bash
# Install all dependencies
npm run install-all

# Start both server + client
npm run dev
```

- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

### Accounts
- Register as **Buyer** to shop
- Register as **Seller** to manage products & view orders
